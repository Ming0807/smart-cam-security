# Skill: Next.js and Vercel

## Use This Skill When

Working on dashboard, API routes, or deployment.

## Rules

- Use Next.js App Router.
- Use Vercel for dashboard and API.
- Do not use Vercel as long-running camera stream proxy.
- Validate API requests.
- Use environment variables.
- Keep service role key server-side only.

## Recommended API Routes

- POST /api/log
- GET /api/status
- POST /api/status
- GET /api/commands
- POST /api/commands
- POST /api/commands/result
- POST /api/telegram/callback
