# Skill: Debugging

## Debugging Process

1. Identify failing component.
2. Read error logs.
3. Reproduce the problem.
4. Check recent changes.
5. Fix smallest possible cause.
6. Test again.
7. Update TROUBLESHOOTING.md.

## Common Problem Areas

- Camera firmware
- Weak power supply
- Wi-Fi timeout
- HC-SR04 echo timeout
- SD card mount failure
- Telegram API error
- Supabase auth error
- Vercel env variable missing

## Rule

Do not rewrite large sections until the exact failure is understood.
